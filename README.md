# devRantCLI
A simple terminal client for [devrant.io](https://www.devrant.io)

#### installation
    $ git clone https://github.com/kurtr/devRantCLI
    $ cd devRantCLI
    $ npm install

#### run
    $ npm start

**IMPORTANT : This is an experimental project and is unlikely to be maintained.**
